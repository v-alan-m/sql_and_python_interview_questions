# Unique Elemments Created
## 1. Tickbox Sidebar Radio Elements

```
Convert the st.selectbox into a vertical list of custom boxes using st.radio. Use CSS flexbox (row-reverse) to hide the default radio circles and create a custom visual tick box anchored to the far right edge. Style the option containers to perfectly match the default Streamlit dark dropdowns (solid background color, specific height/width limits, and rounded corners).
```
